📖 Getting Started

Install and run CodeProbe on any machine with a single command:

npm install -g codeprobe-scanner
npx codeprobe-scanner scan /path/to/repo --fix

That's it! The CLI will:

  • Auto-install Bun if needed
  • Query cloud database for known CVEs
  • Test vulnerabilities locally
  • Generate patches with AI
  • Create GitHub PR with evidence

🌐 Cloud Deployment

Deploy the server to Google Cloud Run:

gcloud run deploy codeprobe-api \ --source . \ --platform managed \ --region us-central1

Cloud URL:
https://codeprobe-901164477360.us-central1.run.app

💻 Local Development

Run the server locally for testing:

bun src/api/server-cli.ts

Then in another terminal:

SERVER_URL=http://localhost:8080 \ CODEPROBE_SECRET=dev-token \ npx codeprobe-scanner scan .

📊 How It Works

  1. Local Scan: Parse dependencies on your machine
  2. Cloud Lookup: Query Firestore for known CVEs
  3. Sandbox Test: Daytona verifies exploitability locally
  4. AI Patch: Kimi generates fixes automatically
  5. Record: VideoDB captures exploit as proof
  6. Create PR: GitHub PR with video evidence
